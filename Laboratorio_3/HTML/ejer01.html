    Seccion etiquetas semanticas estructurales mas importantes

    • HEADER
    • FOOTER
    • SECTION
    • ARTICLE
    • NAV
    • ADDRESS

    Las etiquetas semánticas estructurales estan relacionadas directamente con el significado del contenido y no con estilos ni Las posiciones dentro del documento


    Reglas para Etiquetas semanticas

    • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION
    • Las unidades de contenido se asocian a las etiqueas ARTICLE. O sea que un SECTION anida multiples ARTICLES
    • Las etiqeutas HEADER y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. O sea un FOOTER podría anidar otro HEADER y FOOTER
    • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion dele documento
    • Reservar etiqueta H1 para usar dentro de etiquetas top level como main, header o footer
